# GaLaBau Vertragsanalyse CH

Statische Web-App fuer Vercel, die einen Prompt fuer Werkvertragsanalysen im Schweizer Garten- und Landschaftsbau erzeugt.

## Lokal oeffnen

`index.html` kann direkt im Browser geoeffnet werden.

## Vercel

Das Projekt benoetigt keinen Build-Schritt. Vercel kann den Ordner direkt als statische Site deployen.
